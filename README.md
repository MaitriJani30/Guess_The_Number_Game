
# Number Guessing Game (Python)

A fun, interactive console-based Number Guessing Game built with Python. The computer randomly selects a number between 1 and 100 and your goal is to guess it in as few attempts as possible!

## Features
* **Interactive Gameplay:** Receive instant higher/lower hints after every guess.
* **Attempt Tracking:** Keeps count of how many tries it takes you to find the secret number.
* **Error Handling:** Crash-proof! It gracefully handles invalid inputs (like letters or blank spaces) without breaking the game.
* **Infinite Replayability:** Play as many rounds as you want without having to restart the program.
* **Graceful Exit:** Option to quit mid-game by entering `-1`.

## Prerequisites
To run this game, you will need **Python 3.x** installed on your computer. You can download it from [python.org](https://www.python.org/downloads/).

## How to Run
1. Clone this repository to your local machine:
 ```bash
git clone [https://github.com/MaitriJani30/Guess_The_Number_Game.git](https://github.com/MaitriJani30/Guess_The_Number_Game.git)
 ```

*(Alternatively, you can download the script directly as a ZIP file).*

2. Navigate to the project directory in your terminal:
 ```bash
cd Guess_The_Number_Game

 ```


3. Run the Python script:
 ```bash
python main.py
 ```


*(Note: Depending on your setup, you might need to use `python3` instead of `python`).*

## How to Play

1. The program will prompt you to guess a number between 1 and 100.
2. Type your guess and press `Enter`.
3. The game will tell you if your guess is too high, too low, or spot on.
4. Keep guessing until you find the correct number!
5. To quit at any time during a round, type `-1`.
